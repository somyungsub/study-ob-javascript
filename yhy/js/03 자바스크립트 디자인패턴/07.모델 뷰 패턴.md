## 목적

## 1. MVC 패턴
- 모델: 데이터 담고있는놈
- 컨트롤러: 모델의 상태 업데이트 및 뷰에 전달
- 뷰: HTML 종종 XML, JSON 혹은 데스크톱, 스마트폰의 스크린
